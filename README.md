# Mini Mart Online Stores

### Installation

##### Laravel

`composer install`

##### React

`cd frontend`
`npm install`

-   Features
    1. Buyers
        - Needs Login
        - View Products
        - Add to Cart
        - Buy Products in Cart
        - View Product Status
    2. Admin
        - CRUD Product | D = Soft Delete
        - Update Product Status From Transactions
        - Restock Product
    3. Additional
        - Search
